Oracle Database

Visibilidade completa sobre alterações e acessos no Oracle Database

Simplifique a auditoria do Oracle Database

RELATÓRIOS A NÍVEL DE SERVIDOR

Captura todas as alteraões críticas às configurações do Oracle monitorando SQL statements e privileges, além de outros componentes do servidor, e entrega dados como quem, o que, onde e quando.

MONITORAMENTO DETALHADO

Mantém você atualizado sobre quaisquer alterações feitas na sua base de dados capturando execuções de SQL statements como INSERT, UPDATE e DELETE, nas propriedades, triggers e outros objetos da base de dados.

AUDITORIA DE ACESSOS COMREENSÍVEL

Monitora tentativas bem e malsucedidas na sua base de dados Oracle, assim como tentativas de acesso aos dados, para que você possa rapidamente detectar atividade inapropriada ou maliciosa.

ALERTAS SOBRE PADRÕES DE AMEAÇA

Torna possível que você rapidamente identifique quem tem acesso a quais pastas e arquivos, assim como aos dados expostos. Você também pode facilmente comparar o estado atual de configuração com um ponto anterior.

RELATÓRIOS E DASHBOARDS ACIONÁVEIS

Torna possível que você rapidamente identifique quem tem acesso a quais pastas e arquivos, assim como aos dados expostos. Você também pode facilmente comparar o estado atual de configuração com um ponto anterior.

BUSCA INTERATIVA

Permite que você rapidamente organize os resultados do Oracle e aperfeiçoe seus critérios de busca até que você encontre a causa de um problema ou a resposta de uma questão específica de um auditor. Salve suas buscas como relatórios customizados e receba-os de forma automatizada.
