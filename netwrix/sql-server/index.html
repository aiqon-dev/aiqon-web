Netwrix Auditor para

SQL Server

Visibilidade completa sobre o que acontece no seu Microsoft SQL Server

Simplifique a auditoria do SQL Server

AUDITORIA DE ALTERAÇÕES A NÍVEL DE SERVIDOR

Habilita a auditoria de dados sigilosos provendo informações sobre os tipos de dados que você possui, onde estão armazenados, quem tem acesso a esses arquivos e como estão sendo usados.

RELATÓRIOS RELEVANTES E PERSONALIZÁVEIS

Monitore as alterações feitas no conteúdo das bases de dados de acordo com as configurações de auditoria do SQL Server.

MONITORAMENTO DE LOGON DETALHADO

Facilita a segurança do SQL Server, registrando e alertando sobre tentativas bem e mal-sucedidas de logon, seja com autenticação do Windows ou do SQL Server.

ALERTAS SOBRE PADRÕES DE AMEAÇA

Torna possível uma reação imediata sobre eventos que requerem atenção, notificando acessos não autorizados e alterações em pontos chave, de modo a evitar vazamentos de dados e quedas de sistema.

RELATÓRIOS E DASHBOARDS PRECONFIGURADOS

Acrescenta aos logs do SQL Server, providenciando informações sobre a integridade dos servidores, além de centenas de relatórios predefinidos e customizáveis, permitindo que você encontre facilmente a informação desejada utilizando filtros e funções intuitivas.

RELATÓRIOS DE ANÁLISE DE ARQUIVO

Gera informação detalhada sobre propriedade de dados, utilização e volume desses dados, além de identificar arquivos vencidos e duplicados.
