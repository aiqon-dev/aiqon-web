O que é o Cloud Management Suite?

O CMS é uma suite de ferramentas para gestão de parques de computadores à partir da nuvem.

Direto do seu navegador você pode realizar tarefas de atualização, reiniciar máquinas, extrair relatórios, instalar pacotes de software, acessar qualquer máquina remotamente e muito mais!

São quatro pacotes de ferramentas diferentes, assim você só paga pelo que for utilizar. Temos um plano perfeito para o seu ambiente!
