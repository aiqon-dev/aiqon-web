Scan de vulnerabilidades

A Segurança de Servidores e Endpoints garantida com a eliminação de qualquer brecha que possa oferecer perigo para seus dispositivos

PORT SCANNERS
Determine quais portas TCP / UDP estão abertas e conhecidas por explorar bugs no sistema operacional. Os resultados disso incluirão detalhes do CVE para as ameaças colocadas.
ATAQUES DE FORÇA BRUTA
Usando uma lista de senhas conhecidas, cada usuário local será verificado e marcado se alguma das contas de usuário tiver uma senha em branco ou uma da lista conhecida.
POLÍTICAS DO USUÁRIO DO WINDOWS
Tratado com uma lista de verificações executadas nas contas de usuário local para verificar se não existem vulnerabilidades conhecidas que possam ser expostas facilmente.
PORTAS SNMP
Veja quais portas SNMP estão abertas e conhecidas por explorar bugs no sistema operacional. Os resultados disso incluirão detalhes do CVE para as ameaças colocadas.
SERVIÇOS RPC
Descubra quais serviços RPC estão em execução e que são conhecidos por explorar bugs do SO. Os resultados incluirão detalhes do CVE para as ameaças determinadas.
CONFORMIDADE
Detecte elementos de segurança dos dispositivos que passam ou não nos requisitos do PCI DSS.
