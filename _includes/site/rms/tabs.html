Com o software de classificação de dados da Netwrix, você pode automatizar o processo de identificação de não registros para facilitar sua limpeza, para que todos possam ser mais produtivos no trabalho

A Classificação de dados do Netwrix permite descobrir automaticamente os registros com os quais os usuários comerciais não trabalham mais e movê-los para o seu arquivo morto.
